Welcome to my url shortener App!

Put some url after this url in navigation bar and I will generate a short url that works!

  1. You can pass a URL as a parameter and you will receive a shortened URL in the JSON response.
  2. If you pass an invalid URL that doesn't follow the valid http://www.example.com format, the JSON response will contain an error instead. (you do not need to type 'http://' I have already done this for you :D)
  3. When you visit that shortened URL, it will redirect you to your original link.
